# 狗

这是一只狗，如果你一段时间没有喂它，它就会咬你

## 设计

首次喂狗时将会给狗进行注册，注册你是谁，名字是什么，当前状态是什么。以及告诉狗说，如果多久没有喂就开咬，咬的间隔等等

以后每次喂狗都可以更新自己的状态，以及更新配置属性

如果超过设置的时间没有再次喂狗，则狗将会进行 汪 的警告

## 用途

这就是一个非常简单的监控程序，可以用来做服务的监控。如果服务挂了，一段时间不能进行喂狗了，那狗将会进行 汪 通知给到监控客户端程序

## 部署

这是一个 c-s 模型，需要有一个服务端。服务端最好是一个大家都能访问的服务器。需要有一个或多个客户端，每个客户端都能接收 汪 信息，了解有哪些服务超过时间没有喂狗
